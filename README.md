# getting started


```
cd cmd
go run main.go --server=true
````
run as sudo when running app store
```
cd cmd
go build main.go && sudo ./main --server=true
```

rename the `config-example.yml` file to `config.yml`

run the bash script to build and start
`bash build.bash`

## default port

8080
